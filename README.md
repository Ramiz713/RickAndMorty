# Rick and Morty
Rick and Morty app, based on open API https://rickandmortyapi.com. 
### Used libraries:
* AndroidX
* DI - Dagger 2
* MVP - Moxy
* Async - RxJava 2
* Network - Retrofit 2
* Image loading - Glide
* Unit Test - Mockito
* Android Test - Espresso
* Database - Room
* Navigation - Cicerone
